Хуки жизненного цикла

хук created - можно использовать для выполнения кода после создания экземпляра

mounted - Вызывается сразу после монтирования экземпляра, когда взамен el создан vm.$el. Если корневой экземпляр смонтирован на элемент документа, vm.$el тоже будет элементом документа. Не вызывается при отрисовке на стороне сервера.

update - Вызывается после того, как виртуальный DOM был обновлён из-за изменения данных.
